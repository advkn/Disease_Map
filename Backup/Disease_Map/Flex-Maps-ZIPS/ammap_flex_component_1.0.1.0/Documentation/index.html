Installing components

To use SWC files in your Flex projects, you should add them to the project's library path.

  1. With a project selected in the Flex Navigator view, select Project > Properties > Flex Build Path.
  2. Click on the Library Path tab.
  3. Click on the Add Swc button.
  4. Browse to and select the amMapFB3.swc file if you work with Flex Builder 3. Select amMapFB4.swc if you work with Flex Builder 4.
  5. Click OK
  6. The component is now added. You should see amMap folder in the components window (only available when you are in design mode).
Visit Flex documentation for more options on installing components.

Change log

1.0.1.0

This is important update!

Some changes which might affect previous behavior were made in this version, read the following carefully.

AmMap (FB4 build) now can be used with HERO SDK (except AmMapLegend class). This means you can build mobile applications with amCharts for Android and iPhone now. To accomplish this we had to get rid of classes, not supported by HERO SDK. We made changes both in FB3 and FB4 build so that it would be easier to maintain and document. This also made our components much lighter and a bit faster. Below listed changes which might have influence to your applications with previous versions of amMap.

AmMap and SmallMap classes no longer extends Canvas, they extend UIComponent instead. This means that they do not support styles/properties/methods specific to Canvas anymore. These classes still hav styles backgroundColor and backgroundAlpha styles. This change might cause error on your application if you used Canvas specific styles or properties. If you need them anyway, you can always put your map into Canvas container.

amMap no longer uses mx ProgressBar to show progress of map loading. We build our own progressbar instead. You can still use mx Progressbar, as mapProgressBar accepts any object now.
